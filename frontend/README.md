Dark, minimalist landing + ERP‑style internal app using Material UI.
Talks to the FastAPI backend (`VITE_API_URL`, default: `http://localhost:8000/api/v1`).

## Quickstart

```bash
npm install
VITE_API_URL=http://localhost:8000/api/v1 npm run dev
